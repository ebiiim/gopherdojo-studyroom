# imgconv

```
Usage: imgconv [-r true|false] [-src jpeg|png|bmp|tiff] [-dst jpeg|png|bmp|tiff] DIR
  -dst string
        Destination image format (default "png")
  -r    Recursivly traverse directories (default true)
  -src string
        Source image format (default "jpeg")
```

```
imgconv -dst bmp path/to/images
```
